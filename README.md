# ELT - Agência Nacional de Petróleo

## Objetivo do projeto

Por meio deste projeto resolvi consolidar alguns conhecimentos que venho adiquirindo no dia a dia em meus estudos sobre Engenharia de Dados. Os dados utilizados estão disponívels no Portal de Dados Abertos do Gov BR. Acesso pelo seguinte link: <a>https://www.gov.br/anp/pt-br/centrais-de-conteudo/dados-abertos</a>

Utilizei essa base de dados por ter alguns transformações e limpezas que podem ser feitas para liberar os dados de uma melhor forma para futuras análises.

## Etapas 

> 1. Extração dos arquivos .csv (E)

> 2. Carga em forma bruta em arquivo .parquet (L)

> 3. Análise Exploratória dos Dados

> 4. Transformação dos Dados em novo arquivo .parquet dos dados transformados (T)

## Ferramentas Utilizadas

> Pandas 

> Jupyter
